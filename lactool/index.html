Main features

• Import, export and share maps
• Manage options, roles and materials of maps
• Merge multiple maps
• Manage wallpaper of the in-game cellphone
• View screenshots taken using the in-game cellphone

⚠️ October 2023 security patches

App may not work correctly on Android 11 and above versions using October 2023 or higher security patches as those patches prevent apps from requesting access to files in Android/data folder.
If you are affected by this, you can use following methods until a solution is found:
  • Connect your device to a PC with USB, or use wireless debugging to manually manage files in Android/data/com.MA.LAC/files.
    • editor folder contains maps
    • wallpaper folder contains in-game cellphone wallpapers
    • screenshots folder contains screenshots taken from the in-game cellphone
  • Copy .txt link of the map and paste it on map name field upon creating a map, this will download the map.
  • Copy link of your wallpaper and paste it on in-game cellphone settings.
