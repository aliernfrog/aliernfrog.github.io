Play Store Bot
Google Play Store™️ scraper Discord bot with many useful features

Main features

• Tracking updates of apps
• Getting useful information about apps, including some information which may not be visible in Google Play client.
• Multiple languages: English, Polish and Turkish

Tracking app updates

• App updates can be tracked using /app-track command.
• Only 5 apps can be tracked per server.

Some useful information

• To search apps, use /app-search command.
• To search using app ID, use /app-info command.
• To change language, use /options command.
