Ms. Indinibee (or just Indinibee) is a bee farming themed Discord economy bot.

Currently the bot is pretty simple and does not have much features. Start finding items using /find, sell items you do not need using /shop sell, collect products of your bees using /collect and access shop using /shop view. To get more information about commands, you can use Discord's app launcher.
