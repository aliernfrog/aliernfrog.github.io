Android/data access
Apps were able to easily access Android/data folder by having the user grant access to it.
However, September 2023 update to the Files app made this impossible for Android 12 and above.

The easiest workaround

Uninstalling updates of the Files app lets apps request access to Android/data.

How to uninstall updates of Files app?

Keep in mind that these steps may vary depending on your device
Device settings > Apps > All apps/Manage apps > Files (app name depends on your language) > Uninstall updates

Can't find Files app on the list?

Look for an option which hides the system apps in the list. It might also be in a three-dot menu.

Can't uninstall updates?

Check out the workarounds listed here.

Other workarounds

  • Connect your device to a PC with USB, or use ADB wireless debugging
  • Use Shizuku with:
    • ZArchiver (closed-source) - i couldnt find the option, maybe i missed something?
    • FV File Manager (closed-source)
  • For workarounds for LAC or Polyfield, check this


LAC & Polyfield workarounds

  • LAC:
    • Copy .txt link of the map you want to download and paste it on map name field, then hit create
    • Copy link of the wallpaper you want to import and paste it on in-game cellphone settings
    • Manually manage files, check these first
      • Android/data/com.MA.LAC/files/editor contains your maps
      • Android/data/com.MA.LAC.files/wallpaper contains in-game cellphone wallpapers
      • Android/data/com.MA.LAC.files/screenshots contains screensots taken using in-game cellphone
  • Polyfield:
    • Download maps from in-game maps list
    • Manually manage files, check these first
      • Android/data/com.MA.Polyfield/files/editor contains your maps
