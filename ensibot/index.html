Ensi
A cursed chatting bot

Main features

• Ability to chat with Ensi
• Snipe command
• Various useful commands such as /userinfo

Chatting feature

• This feature can be adjusted or completely disabled by server managers using /chat command.
• By default, Ensi will automatically respond to messages which mentions him, or if it was sent in a thread Ensi is in.
• There are also some random interactions such as washing and feeding Ensi. You can earn Ensipoints when interacting, but they currently don't have any use.

Adding to servers is temporarily disabled
You can still use Ensi on existing servers.
